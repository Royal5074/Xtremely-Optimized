# Installation Guide

## Requirements
- Minecraft (latest compatible version)
- Fabric/Forge mod loader (as required)
- Modrinth or CurseForge launcher

## Installation Steps
1. Download **Xtremely Optimized** from [Modrinth](https://modrinth.com/modpack/xtremely-optimized).
2. Open your **Modrinth App**, **Prism Launcher**, or any supported launcher.
3. Select the modpack and install it.
4. Start Minecraft and enjoy optimized performance!

For post-installation optimization, check the [Post Installation Guide](./post-installation.mdx).
