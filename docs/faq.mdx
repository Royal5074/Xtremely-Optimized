# Frequently Asked Questions

### ❓ What is Xtremely Optimized?
Xtremely Optimized is a performance-based modpack that improves FPS and gameplay.

### ❓ Can I add more mods?
Yes! However, ensure they do not impact performance negatively.

### ❓ Is this compatible with multiplayer?
Yes, it works with multiplayer and optimizes server performance.
