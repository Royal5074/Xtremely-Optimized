# Post Installation Guide

After installing **Xtremely Optimized**, follow these steps for the best experience:

### 1. Adjust Video Settings
- Lower render distance for better FPS.
- Enable **fast rendering** options.
- Disable unnecessary particles.

### 2. Optimize Java Settings
- Allocate 4-6GB RAM for smooth gameplay.
- Use the latest Java 17+ version.

### 3. Ensure Mod Compatibility
- Avoid conflicting mods that impact performance.
- Use only recommended optimization mods.

You're now ready to enjoy the best Minecraft experience!
